HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. Developed by Tim Berners-Lee in 1991, HTML structures content with elements and tags. It defines headings, paragraphs, links, images, and lists, forming the foundation of web development and enabling the creation of structured, interactive, and visually appealing web pages.

It is used to structure web pages.

<html></html>

Elements are the building blocks of HTML.

  • <head>
  • <body>
  • <div>

Attributes provide additional information about elements.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of HTML documents. Developed by the W3C, CSS controls layout, colors, fonts, and overall design, enabling the separation of content and style. This enhances web pages' visual appeal and improves user experience and maintainability.

It is used to style web pages.

body { color: black; }

Selectors are used to select elements.

  • Class selector
  • ID selector
  • Element selector

Properties define the styles applied to elements.

JavaScript

JavaScript is a versatile programming language essential for web development. It enables interactive and dynamic content on web pages, such as animations, form validations, and real-time updates. JavaScript works alongside HTML and CSS, enhancing user experiences by making websites more engaging and responsive.

It is used to make web pages interactive.

function myFunction() { alert('Hello, World!'); }

JavaScript can be used to manipulate the DOM.

  • Variables
  • Functions
  • Events

JavaScript can interact with APIs.
